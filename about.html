More About Me

I am currently a graduate student at University of Texas at Dallas studying Computer Science. After graduating from Baylor University with a bachelors in mathematics and business, I moved to Plano, Texas. Since then I have been on a self guided curriculum, learning about machine learning algorithms, computer vision, and software architecture.

I am currently focusing my studies on the recent advancements in computer vision utilizing deep learning. Particularly the use of convolutional neural networks for video understanding. Over the next few months I hope to implement convnets in RESTful web services and drone flight control programs.

Any Questions?

Why did you make this website?
I built this website to share my projects and motivations with others. In doing this, I hope to show my experience and interest in computer vision and web application development. As an added benefit of making this site, I was able to develop my front-end web design skills in setting the layout, as well as my back-end development skills by hosting this site on DigitalOcean.
How long have you been practicing machine learning?
I discovered machine learning a year ago after graduating from Baylor University. Since then I have immersed myself in the data science and machine learning communities. Spending the last few months learning about the mathematical and statistical foundations of the field, as well as implementing the algorithms in personal projects.
Do you have any experience with database management?
Though I have had experience with scraping, cleaning, and storing datasets locally, I have not had exposure to any of the major database systems. I intend to work more with MySQL and MongoDB as soon as I come to a project that would benefit from having used these tools.
Why are you interested in computer vision / machine learning?
Everyday I read a new Medium post about a business that was able to use machine learning or computer vision to provide value for their clients. Despite the movement towards integrating AI into existing technologies, the implementation of machine learning (especially computer vision) is still in its infancy. This field is the wildwest, and I want to work with the pioneers and innovators that create new and amazing things.
What industry are you looking to work in?
I am very interested in web application development. I am drawn towards the industry's focus on rapid improvement, innovation, and team-oriented creativity. Furthermore, I think when it comes to improving existing technologies there is a great deal of low-hanging fruit to be reached thanks to the movement towards microservice architecture and docker containerization. These developments make machine learning more integrable than ever.
What kind of work do you see yourself doing?
I am interested in building modular machine learning applications that can communicate with the backend architectures of existing applications. Position titles that do this work are not well defined, so I like to describe my perfect future position to be a mix of data science, machine learning engineering, and devops. Essentially a computer vision specialist with the skills to deploy a model application to production.
